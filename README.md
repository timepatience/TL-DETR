## Overview

TL-DETR is a high-performance tunnel leakage detection framework designed for low-contrast laser-scanned tunnel imagery.

It addresses key challenges in tunnel defect detection, including:

- weak leakage signatures in low-contrast tunnel environments,
- background interference from cables, pipelines, lining joints, and concrete textures,
- fragmented detection of large-span and elongated leakage defects,
- efficient deployment requirements on resource-constrained edge devices.

The framework integrates:

- a hybrid cascade-residual backbone for enhanced long-range defect topology modeling,
- magnitude-rectified linear attention for recovering weak leakage responses,
- spatial-shift feature mixing upsampling for precise boundary localization,
- structured distribution alignment distillation for lightweight model deployment.
